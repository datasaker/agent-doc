현재 Kubernetes 환경에 `DataSaker`의 선행 작업이 진행되지 않으셨다면 `DataSaker` 선행 작업을 먼저 진행하여 주시기 바랍니다. [DataSaker 선행 작업](${PREPARATION_MANUAL_KR})

# Mongo agent 설치하기
## 1. Mongo agent 설정값 등록
```shell
cat << EOF >> ~/datasaker/config.yaml

mongoAgent:
  list: []
EOF
```
