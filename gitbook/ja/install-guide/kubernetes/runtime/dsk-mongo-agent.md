# dsk-mongo-agent

## Kubernetes環境でDataSaker Mongo agentをインストールする

## DataSaker先行作業を進めましたか？

現在Kubernetes環境で `DataSaker`の先行操作が進行していない場合は、 `DataSaker`先行操作を先に進んでください。 [DataSaker先行操作](README.md)

## Mongo agentのインストール

### 1. Mongo agent 設定値登録
```shell
cat << EOF >> ~/datasaker/config.yaml

mongoAgent:
  list: []
EOF
```
